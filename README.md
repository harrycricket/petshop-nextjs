# PET SHOP - Next.js

This is a shopping web application built with Next.js. The application includes several features:

## Features

### Authentication
- Login
- Register
- Logout

### Account
- Get user information
- Update user information

### Product
- Add new products
- Update existing products
- Delete products
- Get a list of all products

### Media
- Upload images

### Test API
- API endpoints for testing purposes

## Technologies Used

### Front End
- Next.js
- Correct_UI_Library
- Tailwind
- TypeScript
- SCSS/ CSS

### Back End
- Node.js
- Fastify
- SQLite
